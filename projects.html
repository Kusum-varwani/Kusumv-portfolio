A one-page static website is a simple and efficient type of website that consists of a single HTML page. All of the content, including text, images, and other media, is displayed on this single page. Static websites are often used for portfolios, landing pages, small business sites, or event promotions.

This simple project demonstrates the use of JavaScript to dynamically change the background color of a webpage. The project involves a button that, when clicked, generates a random RGB color and applies it as the background color of the page.The goal of this project is to showcase basic DOM manipulation,event handling, and random number generation to achieve a visually interactive feature on a webpage.

Technologies: HTML: Structuring the content with a simple button to trigger the action. CSS: Styling the button and centering it within the page for a clean and simple user interface. JavaScript: Handling the event listener for the button click, generating random color values, and applying the color to the page.
